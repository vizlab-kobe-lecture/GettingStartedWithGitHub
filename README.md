1. アカウントの作成
まずは、GitHubアカウントを作成します。GitHubのホームページにアクセスし、以下の手順でアカウントを作成してください。

1. プロジェクトのクローニング
vizlab-kobe-lectureに登録されているリポジトリを編集用にローカルPC環境にコピー（クローン）します。以下の手順で、テスト用リポジトリ（Test）を各自の環境にクローンしてください。
